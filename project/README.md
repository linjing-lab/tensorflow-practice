# 直肠癌肿瘤识别

## 目录

```text
- Data (处理后的数据集)
    - X.npy (特征数据集)
    - y.npy (标签数据集)
- Logs (训练日志：_训练集大小_批次量_批量大小)
    - training_5000_20_128.log
    - training_6000_20_128.log
- Mask (原始数据集)
- Models (最优模型：_训练集大小_批次量_批量大小)
    - best_model_5000_20_128.h5
    - best_model_6000_20_128.h5
- data_pretreat.ipynb (用于数据预处理)
- model_train.ipynb (用于模型训练与评估)
```
